## Core Web Vitals: The Foundation of Performance

Modern web performance is measured through Core Web Vitals:

1. Largest Contentful Paint (LCP): Loading performance
2. First Input Delay (FID): Interactivity
3. Cumulative Layout Shift (CLS): Visual stability

<Steps>
### Optimizing Loading Performance

Focus on reducing your LCP score with these techniques:

```typescript
// Use Image component for automatic optimization
import Image from "next/image";

export function OptimizedImage() {
  return (
    <Image
      src="/hero.jpg"
      alt="Hero"
      width={1200}
      height={600}
      priority // For above-the-fold images
      quality={85} // Balance quality and size
    />
  );
}
```

### Improving Interactivity

Reduce JavaScript bundle size to improve FID:

```typescript
// Before: Importing entire library
import { format, addDays, subDays } from "date-fns";

// After: Import only what you need
import format from "date-fns/format";
import addDays from "date-fns/addDays";
```

### Eliminating Layout Shifts

Prevent CLS with proper image and font handling:

```typescript
// Preload critical fonts
<link
  rel="preload"
  href="/fonts/inter.woff2"
  as="font"
  type="font/woff2"
  crossOrigin="anonymous"
/>

// Reserve space for dynamic content
<div style={{ aspectRatio: '16/9' }}>
  <Image fill src="/banner.jpg" alt="Banner" />
</div>
```
